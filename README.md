### 폴더 구조

- /components : 각 페이지 별 컴포넌트 모음
- /components/common : 공통으로 사용할 컴포넌트(Header.js, Footer.js, GlobalStyles.js 등)
- /redux/modules : action.js, middlewarej.js, reducer.js, types.js 별로 각각 나누어 정리
